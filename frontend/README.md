# Iris Classifier - Web Frontend

React web frontend with iPhone-style design for the Iris Classifier API.

## Features

- 🎨 iPhone-style UI with Tailwind CSS
- 🎚️ 4 Interactive sliders for flower measurements
- 📊 Real-time prediction results with probabilities
- 🔌 Axios API client integration
- 📱 Responsive design

## Setup

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Environment

Set API URL:
```bash
export VITE_API_URL=http://localhost:8080
```

## Technologies

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Axios
